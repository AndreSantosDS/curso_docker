Podemos nomear a imagem que criamos;
Vamos utilizar o comando docker tag <id da imagem> para isso;
Também podemos modificar a tag, que seria como uma versão da imagem, semelhante ao git;
Para inserir a tag utilizamos: docker tag <nome>:<tag>

docker tag 6fd8c7003738 minhaimagem
docker tag 6fd8c7003738 minhaimagem:minhatag -> faz uma copia da imagem e troca o nome da tag que era latest para minhatag
docker tag --help

o comando docker tag ele utilizar o id da imagem para poder nomear e inserir uma tag
